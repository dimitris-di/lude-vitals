# Third-party Notices

LudeVitals currently has no vendored third-party source code and no package-manager dependencies outside Apple's platform frameworks and Swift toolchain.

The project acknowledges community-documented sensor access techniques from projects such as Stats, asitop, and iStatistica in [README.md](README.md#acknowledgments). Those projects are not bundled with LudeVitals.

If future releases add third-party code, dependencies, or redistributed assets, this file should list their names, upstream URLs, licenses, and any required notices.
