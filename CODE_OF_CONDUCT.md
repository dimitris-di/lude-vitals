# Code of Conduct

## Our pledge

We want LudeVitals to be a welcoming project for contributors of every background and experience level. We pledge to make participation a harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive in discussions and code review.
- Assume good faith. Ask before assuming.
- Focus on what is best for the project and its users.
- Keep technical disagreement technical.

## Unacceptable behavior

- Personal attacks, insults, or derogatory comments.
- Harassment in any form, public or private.
- Publishing others' private information without explicit permission.
- Conduct that would be inappropriate in a professional setting.

## Enforcement

If you experience or witness behavior that violates this code, please report it to **demetrisd25@gmail.com**. Reports will be reviewed and addressed privately and promptly. Maintainers who violate the code lose their commit access until the issue is resolved.

## Scope

This code applies to all project spaces — issues, pull requests, discussions, commit messages, and anywhere else contributors interact about the project.

This document is inspired by the Contributor Covenant but kept intentionally short. For the full Contributor Covenant text, see https://www.contributor-covenant.org/.
